# Insites Sidebar Footer Menu```<ins-sidebar-footer-menu>```

### Versions
1.0.23

### Basic Features
Displays sidebar footer items.


### Configuration Options

| ATTRIBUTE | TYPE | DEFAULT | OPTIONS | DESCRIPTION |
| ------- | ------- | ------- | ------- | ------- |
| icon | string | "icon-settings-1" | icon-close-1, icon-utilities, icon-lock-1, icon-logout-1 | displays icon of sidebar item |
| label | string | 'Settings' | any | Displays label of sidebar footer menu |
| withSubmenu | boolean | false | true, false | enables toggle menu for sidebar footer menu |

### Usage Details
```
<ins-sidebar-footer-menu icon="icon-dashboard" label="Dashboard" withSubmenu></ins-sidebar-footer-menu>
```

### Default Values
```
icon = "icon-settings-1"
label = "Settings"
menuToggled = false
```


### Browser Support
- Chrome (and all Chromium based browsers)
- Safari
- Firefox
