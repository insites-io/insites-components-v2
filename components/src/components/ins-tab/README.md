# Insites Tab ```<ins-tab>```

### Versions
1.0.0

### Basic Features
Creates a tab container. This is just the wrapper used to hold the tab headers and contents.

   
### Usage Details
```
<ins-tab>
  <ins-tab-item active label="Tab 1"> Tab 1 HTML content </ins-tab-item>
  <ins-tab-item label="Tab 2"> Tab 2 HTML content </ins-tab-item>
  <ins-tab-item label="Tab 3"> Tab 3 HTML content </ins-tab-item>
</ins-tab>
```

### Browser Support
- Chrome (and all Chromium based browsers)
- Safari
- Firefox
