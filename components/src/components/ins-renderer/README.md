# Insites Renderer ```<ins-renderer>```

### Versions
1.0.22

### Basic Features
Renders an iframe.

### Configuration Options

| ATTRIBUTE | TYPE | DEFAULT | OPTIONS | DESCRIPTION |
| ------- | ------- | ------- | ------- | ------- |
| link | string | undefined | any | add route anchor for iframe |
| app | boolean | false | true, false | render a iframe |
| label | string | undefined | any | Defines for an element |

### Usage Details
```
<ins-renderer></ins-renderer>
```

```
N/A
```
