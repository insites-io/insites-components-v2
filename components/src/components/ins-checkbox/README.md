# Insites Checkbox ```<ins-checkbox>```

### Versions


### Basic Features
Defines a checkbox.

### Configuration Options

| ATTRIBUTE | TYPE | DEFAULT | OPTIONS | DESCRIPTION |
| ------- | ------- | ------- | ------- | ------- |
| checked | boolean | undefined | true, false | Specifies if checkbox is activated or not |
| value | string | undefined | any | Specifies value of checkbox |
| label | string | undefined | any | Displays label of button |


### Usage Details
```
<ins-checkbox label="test" checked></ins-checkbox>
```
