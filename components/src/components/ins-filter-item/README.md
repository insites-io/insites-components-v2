# Insites Filter Item ```<ins-filter-item>```

### Versions


### Basic Features
Displays dropdown items of ins-filter

### Configuration Options
| ATTRIBUTE | TYPE | DEFAULT | OPTIONS | DESCRIPTION |
| ------- | ------- | ------- | ------- | ------- |
| name | string | 'Category Label' | any | Displays name of filter-items |
| options | any  | '["Category 1", "Category 2", "Category 3"]' | any | Displays item options |

### Usage Details
```
<ins-filter>
    <ins-filter-item name="Operating System" options='["Windows", "macOS", "Linux"]'></ins-filter-item>
</ins-filter>
```

### Default Values
```
name = 'Category Label'
options = '["Category 1", "Category 2", "Category 3"]' 
```

### Browser Support
- Chrome (and all Chromium based browsers)
- Safari
- Firefox
