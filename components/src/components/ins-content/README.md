# Insites Content ```<ins-content>```

### Versions
1.0.6

### Basic Features
Displays content of admin page. parent component is ins-admin.

### Configuration Options

| DECORATOR | COMPONENT | TYPE | USAGE |
| ------- | ------- | ------- | ------- |
| @Element() | insAdminEl | Element | Renders content page inside in ins-admin |

### Usage Details
```
<ins-content></ins-content>
```

### Browser Support
- Chrome (and all Chromium based browsers)
- Safari
- Firefox
