# Insites Searchbar```<ins-searchbar>```

### Versions


### Basic Features
Displays searchbar.


### Configuration Options

| ATTRIBUTE | TYPE | DEFAULT | OPTIONS | DESCRIPTION |
| ------- | ------- | ------- | ------- | ------- |
| placeholder | string | undefined | any | specifies a short hint that describes the expected value of an input field  |
| value | string | undefined | any | defines value of searchbar |
| isSearching | boolean | false | true, false | Specifies loading feature of searchbar |

### Usage Details
```
<ins-searchbar placeholder="sample"></ins-searchbar>
```

```
placeholder = 
value = 
isSearching = 
```
