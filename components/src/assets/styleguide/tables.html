Attributes

FIELD ATTRIBUTE TYPE DEFAULT OPTIONS DESCRIPTION
without-search boolean false true, false Removes searchable input field
static-table boolean false true, false Enables table accept static data
loading-screen boolean false true, false Enables loader in table
loader-title any any Defines title of loader
loader-message any any Defines message of loader
loader-icon any any Defines icon of loader
searchbar-placeholder string "" any Defines the short hint that describes the expected value in the input field
heading string "" any Defines header of table
currency string "" any Defines currency in table
table-headers Array of InsBaseTableHeader [] any Defines header of data columns in table
table-data Array [] any Defines data in table (Property keys should match the column header)
page-number number 1 any Defines number of pages in table
page-size number 10 any Defines size of pages in table
page-size-options number 10 any Defines size options of pages in table
total-count any 0 any N/A
bulk-actions any [] any Defines options of Bulk actions dropdown
row-actions any [] any Defines options of rows actions
selected-rows any [] any Defines options of selected rows
empty-value string "" any N/A
sort-order boolean false true, false Enables sorting by order of data tables
sort-keyword string "" any Defines keyword to be sorted

pointer-events

EVENT OBJECT DESCRIPTION
onPaginationChange Gets triggered when paginations are clicked
onTableSearch Gets triggered when an element gets user search input
onTableSort Gets triggered when a table header is clicked
onTableBulkAction Gets triggered when bulk action button is clicked
onTableRowAction Gets triggered when a row action button is clicked

InsBaseTableHeader Properties

KEY DEFAULT OPTIONS DESCRIPTION
label "" Defines a label to display on the table
type string string,number,currency Defines the type of data the column will display
sortable false true/false Defines if the column is sortable
hasTag false true/false Defines if the column data is a tag
hasColumnAction false true/false Defines if the column should also have the row actions on itself
hasImage false true/false Defines if the column will have an image to display, if set to true and column does not have an image it will use the column data's initials
