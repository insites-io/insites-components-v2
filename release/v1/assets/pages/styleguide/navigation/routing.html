New Tab

If your links can be opened in a new tab from an iframe, its going to be different.

It requires three things for the url:

  • The formatted url from the admin, e.g.
  • reroute (Query String) - the route/link you want the page to go to, must be first on the query string
  • reroutelabel (Query String) - the label you want the page to display.
